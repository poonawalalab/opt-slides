Lagrangian and Dual Functions

The Lagrangian

Primal Optimization problem:

min𝐱f(𝐱)subject to 𝐡(𝐱)=0𝐠(𝐱)≥0\begin{align} \min_{\bm{x}} & f(\bm{x}) \\ \text{subject to } & \bm{h}(\bm{x}) = 0\\ & \bm{g}(\bm{x}) \geq 0 \end{align}

Introduce the Lagrangian associated with the problem, defined as

ℒ(𝐱,𝛌,𝛍)=f(𝐱)−𝛌⊤𝐡(𝐱)−𝛍⊤𝐠(𝐱)\begin{align} \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu}) = f(\bm{x}) - \bm{\lambda}^\top \bm{h}(\bm{x}) - \bm{\mu}^\top \bm{g}(\bm{x}) \end{align}

Let 𝐡(𝐱)=0𝐠(𝐱)≥0,𝛍≥0,\begin{align} \bm{h}(\bm{x})&=0 && \bm{g}(\bm{x})&\geq0, && \bm{\mu}&\geq 0,\end{align} then

ℒ(𝐱,𝛌,𝛍)≤f(𝐱) \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu}) \leq f(\bm{x})

The Lagrangian

ℒ(𝐱,𝛌,𝛍)≤f(𝐱) \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu}) \leq f(\bm{x})

Take the infimum of both sides

inf𝐱∈Ωℒ(𝐱,𝛌,𝛍)≤inf𝐱∈Ωf(𝐱)=p⋆ \inf_{\bm{x}\in \Omega} \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu}) \leq \inf_{\bm{x} \in \Omega} f(\bm{x}) = p^\star

Dual function

q(𝛌,𝛍)=inf𝐱∈Ωℒ(𝐱,𝛌,𝛍)q(\bm{\lambda},\bm{\mu}) = inf_{\bm{x}\in \Omega} \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu})

Lower bound property

q(𝛌,𝛍)≤p⋆ q(\bm{\lambda},\bm{\mu}) \leq p^\star

for any dual feasible pair (𝛌,𝛍)(\bm{\lambda},\bm{\mu})

Dual feasibility

The pair (𝛌,𝛍)(\bm{\lambda},\bm{\mu}) is dual feasible if 𝛍≥0\bm{\mu} \geq 0 and q(𝛌,𝛍)>−∞q(\bm{\lambda},\bm{\mu}) > -\infty

Convexity of the dual

The dual problem is a convex optimization problem

Proof

By definition, q(𝛌,𝛍)q(\bm{\lambda},\bm{\mu}) is the pointwise infimum of affine functions of 𝛌,𝛍\bm{\lambda},\bm{\mu}, and is therefore concave. 𝛍≥0\bm{\mu} \geq 0 is an affine constraint. Hence, the dual problem is maximization of a concave function with affine constraints, which is a convex optimization problem.

Duality Gap

Let d⋆=max𝛌,𝛍≥0q(𝛌,𝛍)d^\star = \max_{\bm{\lambda},\bm{\mu}\geq 0} q(\bm{\lambda},\bm{\mu})

Clearly,

d⋆≤p⋆(weak duality) d^\star \leq p^\star \quad \text{(weak duality)}

For convex problems that satisfy appropriate conditions, like Slater’s conditions1, d⋆=p⋆d^\star = p^\star

Generally, the duality gap is p⋆−d⋆p^\star - d^\star, which is non-negative

Local Duality

The dual function q(𝛌,𝛍)q(\bm{\lambda},\bm{\mu}) is defined using a constrained optimization problem:

q(𝛌,𝛍)=inf𝐱∈Ωℒ(𝐱,𝛌,𝛍)q(\bm{\lambda},\bm{\mu}) = inf_{\bm{x}\in \Omega} \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu})

q̂(𝛌,𝛍)=inf𝐱ℒ(𝐱,𝛌,𝛍) \hat q(\bm{\lambda},\bm{\mu}) = inf_{\bm{x}} \mathcal L(\bm{x}, \bm{\lambda}, \bm{\mu})

Local duality says ‘yes’ in some cases.

We assume that the Hessian 𝐋(𝐱*)\bm{L(\bm{x}^\ast)} is positive definite. (We mean that 𝐋(𝐱*)\bm{L(\bm{x}^\ast)} on the whole space ℝn\mathbb{R}^n, not just on the subspace MM.)

Local Duality Theorem

Suppose that the problem minimizef(𝐱)subject to𝐡(𝐱)=𝟎 \begin{align} \operatorname{minimize} & f(\bm{x}) \\ \text{subject to} & \bm{h}(\bm{x}) = \bm{0} \end{align} has a local solution at 𝐱*\bm{x}^\ast with corresponding value r*r^\ast and Lagrange multiplier 𝛌*\bm{\lambda}^\ast. Proof

It is clear that 𝐱*\bm{x}^\ast corresponds to 𝛌*\bm{\lambda}^\ast in the definition of ϕ\phi. Partial Duality


  • It is not necessary to include the Lagrange multipliers of all the constraints of a problem in the definition of the dual function.
  • In general, if the local convexity assumption holds, local duality can be defined with respect to any subset of function constraints.
    • For example, in problem Equation 9 we might define the dual with respect to only the equality constraints ϕ(𝛌)=min𝐠(𝐱)≥𝟎{f(𝐱)−𝛌⊤𝐡(𝐱)}, \phi(\bm{\lambda}) = \operatorname{min}_{\bm{g}(\bm{x}) \geq \bm{0}} \left\{f(\bm{x}) - \bm{\lambda}^\top \bm{h}(\bm{x}) \right\}, where the minimum is taken locally near the solution 𝐱*\bm{x}^\ast but constrained by the remaining constraints 𝐠(𝐱)≥𝟎\bm{g}(\bm{x}) \geq \bm{0}.
  • Again, the dual function defined in this way will achieve a local maximum at the optimal Lagrange multiplier 𝛌*\bm{\lambda}^\ast.
  • The partial dual is especially useful when constraints 𝐠(𝐱)≥𝟎\bm{g}(\bm{x}) \geq \bm{0} are simple such as 𝐱≥𝟎\bm{x} \geq \bm{0} or in a box where many efficient algorithms are available.
    • Steepest descent projection, interior ellipsoidal-trust region methods, etc.
