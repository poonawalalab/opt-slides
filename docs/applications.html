Given joint angles ùê™={qi}\mathbf q = \{q_i\} we can predict the end effector pose ùê±=(x,y,Œ∏)\mathbf x = (x,y,\theta)

Forward & Inverse Kinematics

  • The Forward Kinematics problem ùê±=f(ùê™)\mathbf x = f(\mathbf q) combines known closed-form expressions for individual homogenous transformations
  • Computing the inverse ùê™=f‚àí1(ùê±)\mathbf q = f^{-1}(\mathbf x), however, is not as easy
  • The inverse kinematics problem is often not even unique, which has algorithmic implications

IK Approaches

Since we know how to build f(ùê™)f(\mathbf q), we arrive at two approaches to inverse kinematics

  • Analytic approaches: Build the closed-form expression f(ùê™)f(\mathbf q) and define a closed-form inverse f‚àí1(ùê™)f^{-1}(\mathbf q)
  • Numerical approaches: Numerically search for values of ùê™\mathbf q so that f(ùê™)=ùê±f(\mathbf q) = \mathbf x, where the function ff is either closed-form or numerical

Numerical IK Approach

  • solve optimization: minùê™‚Äñùê±‚àíf(ùê™)‚Äñ22\min_{\mathbf q}\quad \lVert \mathbf x - f(\mathbf q) \rVert_2^2
  • We can add constraints that make the solution unique, or other benefits
